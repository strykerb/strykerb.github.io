Program Overview:

This program was meant to simulate controling a 3D avatar. It implements wireframe, flat shading, and smooth shading, building upon program 3's 3D scene. It allows the user to Modify lighting settings, toggle light sources, and move the camera throughout the scene. It also plays a jump animation.

User Guide

  • [W][A][S][D]: Camera Movement
  • [Mouse]: Camera Paning/Tilting
  • [SPACE]: Jump (Animation)
  • [E]/[Q]: Zoom In/Out
  • [R]: Toggle Orthographic View
  • [1]: Smooth Shading
  • [2]: Flat Shading
  • [3]: Wire Frame
  • [4]: Toggle Diffuse Lighting
  • [5]: Toggle Ambient Lighting
  • [6]: Toggle Specular Lighting
  • [7]: Toggle Point Light
  • [8]: Toggle Directional Light


Implementation Notes

The program was based off of Lucas Nascimento Ferreira's base code. Smooth Shading was achieved using Gouraud Shading. Player jump animation is a quadratic curve. Camera is locked to one height to emulate playing as a charatcer in a video game. Walking input is smooth and continuous, and allows for diagonal movement.
